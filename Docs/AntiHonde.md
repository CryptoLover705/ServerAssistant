# AntiHonde Help

# antihonde
 - Usage: `[p]antihonde [true_or_false=None] `
 - Restricted to: `ADMIN`
 - Checks: `server_only`

Toggles automatically banning H0nde when they join the server.

# purgehonde
 - Usage: `[p]purgehonde `
 - Restricted to: `ADMIN`
 - Cooldown: `1 per 3600.0 seconds`
 - Checks: `server_only`

Ban all "H0nde" alt accounts in this server.
