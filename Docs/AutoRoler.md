# AutoRoler Help

AutoRoler

# autorole
 - Usage: `[p]autorole `

Autorole commands

## autorole disable
 - Usage: `[p]autorole disable `

Disable autorole

## autorole add
 - Usage: `[p]autorole add <role> `

Add a role to be assigned to all new joins

## autorole enable
 - Usage: `[p]autorole enable `

Enable autorole

## autorole remove
 - Usage: `[p]autorole remove <role> `

Remove a role from the autorole list

## autorole list
 - Usage: `[p]autorole list `

List all roles in the autorole list
